#移动端轻量级无缝轮播图
###特点

JavaScript仅100行的轻量级无缝轮播图，不依赖任何框架，面向对象，麻雀虽小，五腑俱全。

###使用方法：

```
new Slider('#sliderWrap', {
     auto: 1, //是否自动播放 1或0
     interval: 3000, //停顿时间
     effect: 'ease-in' //运动效果，贝塞尔曲线
 });
```
